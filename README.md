## 📖 Overview

Smart Task Manager is a productivity-focused web application that helps users efficiently manage daily activities through an intuitive and responsive interface.

The project was developed to strengthen practical skills in:

* Frontend Development
* JavaScript DOM Manipulation
* Responsive UI Design
* Firebase Integration
* Git & GitHub Deployment
