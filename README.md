# 🌸 Smart Task Manager

A modern and responsive task management web application that helps users organize tasks efficiently with Firebase-powered authentication and cloud storage.

## Features

- User Authentication (Signup/Login)
- Add, Edit, Delete Tasks
- Task Categories
- Priority Levels (High, Medium, Low)
- Due Dates
- Search Tasks
- Dark Mode
- Progress Tracking
- Task Statistics
- PDF Export
- Responsive Design
- Firebase Firestore Database

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Firebase Authentication
- Firebase Firestore
- jsPDF

## Future Improvements

- Drag & Drop Tasks
- User Profiles
- Task Reminders
- Notifications
- Real-time Collaboration
